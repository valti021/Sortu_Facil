# 🎯 SortuFácil

Plataforma web desenvolvida para facilitar a **gestão de rifas e jogos online**, tornando o processo mais simples, organizado e interativo.

🔗 Acesse a versão atual:
👉 https://sortufacil.free.nf



## 📖 Sobre o projeto

O **SortuFácil** é um projeto pessoal iniciado em **dezembro de 2025**, com o objetivo de resolver um problema comum: a dificuldade na organização e gerenciamento de rifas.

Com o tempo, a ideia evoluiu para algo maior — uma plataforma completa capaz de suportar **múltiplos tipos de jogos**, oferecendo uma experiência mais prática e divertida para os usuários.

> ⚠️ Importante: Este projeto **não é uma casa de apostas**.



## 🚀 Objetivo

Criar uma plataforma onde o usuário possa:

* Criar jogos (rifas, bingos, etc.)
* Gerenciar participantes
* Controlar todo o processo do início ao fim
* Automatizar sorteios
* Ter uma experiência simples e intuitiva



## 🎮 Jogos planejados

Atualmente, o projeto está em desenvolvimento com foco nos seguintes jogos:

* 🎟️ Rifa
* 🔢 Bingo

Novos tipos de jogos poderão ser adicionados futuramente.
